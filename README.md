## Objectivo

Desenvolver uma calculadora que, ao introduzir uma expressao aritmetica em notacao polonesa inversa, seja capaz de effectuar o calculo e responder com o resultado.

_Exemplos:_

```
Input:  2,3+
Output: 5
```

```
Input: 2,6*3/
Output: 4
```
